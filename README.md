# Relationship Agreement

A beautiful, interactive web-based relationship agreement template for partners to define their mutual commitments, boundaries, and shared values.

## Features

- **Elegant Design** - Minimalist aesthetic with serif typography and subtle borders
- **Interactive Form Fields** - Editable inputs for partner names, agreement terms, and dates
- **Dynamic Content** - Add custom items to any section
- **Print-Ready** - Optimized styling for printing
- **Responsive** - Works on desktop and mobile devices

## Sections

1. **Values & Foundations** - Core principles guiding the relationship
2. **Communication** - How partners will communicate and check in
3. **Boundaries & Needs** - Personal space and emotional boundaries
4. **Shared Life & Responsibilities** - Household, financial, and life decisions
5. **Conflict Resolution** - How to handle disagreements constructively
6. **Custom Terms** - Add your own personalized agreements

## Usage

1. Open `index.html` in any modern web browser
2. Fill in partner names
3. Customize agreement terms by clicking on editable fields
4. Add additional items using the "+ Add item" buttons
5. Set the agreement date and optional review date
6. Print or save as PDF when complete

## Technical Details

- Single-file HTML with embedded CSS and JavaScript
- No external dependencies (besides Google Fonts)
- Auto-populates today's date and a one-year review date
